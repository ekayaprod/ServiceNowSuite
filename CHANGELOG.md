## [Unreleased]

### Features
- ✅ New: Queue Interceptor for monitoring and claiming unassigned tickets automatically.
- ✅ New: Extract onboarding tickets specifically via CSV Extractor builder.
- ✅ Update: Fluid visual UI across elements, improved touch targets, and better semantic boundaries.

## [0.1.0] - 2026-02-11

### Features
- ✅ New: Ticket Template Builder with two-click mode and dependent field support.
- ✅ New: Deferred Bulk Updater for scheduled bulk comment updates.
- ✅ New: Universal Automation Builder for complex rule-based automation with batch processing.
- ✅ New: CSV Data Extractor for exporting list view data with custom filters.
- ✅ New: Environment Detection system for robust ServiceNow execution context handling (Shadow DOM & Iframes).

### Chores
- 🔧 Chore: Added comprehensive documentation including README, ROADMAP, and CONTRIBUTING guides.
- 🔧 Chore: Added verification script and assets.
- 🔧 Chore: Add automated Playwright tests for showResult DOM manipulation.
